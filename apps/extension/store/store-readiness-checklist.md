# Store Readiness Checklist

## Chrome Web Store

- Conta de publisher registrada
- 2-step verification ativa
- Chrome Web Store API habilitada no Google Cloud
- OAuth consent screen criada
- OAuth client criado
- Refresh token gerado
- Item criado no dashboard
- Store listing preenchida
- Privacy preenchida
- Test instructions preenchidas

## Firefox AMO

- Conta AMO criada
- API key JWT emitida
- `browser_specific_settings.gecko.id` validado
- Metadata JSON revisado
- Canal escolhido: `listed` ou `unlisted`

## Public assets do produto

- Homepage publica no ar
- Pagina de privacidade publica no ar
- Pagina de suporte publica no ar
- README atualizado com fluxo de usuario final
- Release artifacts atuais gerados e auditados
